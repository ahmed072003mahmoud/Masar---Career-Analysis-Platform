
# منصة مسار للتحليل المهني | Masar Career Platform

## نظرة عامة (Overview)
منصة مسار هي نظام متطور لتحليل المسارات المهنية باستخدام تقنيات الذكاء الاصطناعي (Gemini AI). تهدف المنصة إلى مساعدة الأفراد في اكتشاف مهاراتهم الحقيقية، ومطابقتها مع اتجاهات سوق العمل الفعلية، وتقديم توصيات مهنية دقيقة مبنية على البيانات.

---

## الميزات التقنية (Technical Features)
- **تحليل الذكاء الاصطناعي**: دمج Gemini API لتقديم توصيات مخصصة.
- **نظام التلعيب (Gamification)**: نظام نقاط (XP)، مستويات، وأوسمة رقمية هندسية.
- **تتبع التقدم**: خط زمني تفاعلي (Timeline) يسجل كافة الإنجازات المهنية.
- **تحليل السوق**: محاكاة حية لبيانات الرواتب، معدلات النمو، والمهارات المطلوبة باستخدام Recharts.
- **إدارة الملف الشخصي**: لوحة تحكم شاملة مع إمكانية تصدير السيرة الذاتية بصيغة PDF.
- **الأداء العالي**: استخدام Lazy Loading لتقسيم الكود وتحسين سرعة التحميل.

## دليل البدء السريع (Quick Start Guide)

### المتطلبات (Prerequisites)
- متصفح ويب حديث يدعم ES Modules.
- مفتاح API صالح لـ Google Gemini (يتم حقنه تلقائياً في البيئة).

### طريقة التشغيل (Running the Project)
1. افتح المجلد الرئيسي للمشروع.
2. تأكد من تشغيل خادم التطوير (Vite).
3. المنصة تعتمد على HashRouter، لذا يمكنك التنقل عبر الروابط العلوية.

## هيكل المجلدات (Project Structure)
```
/
├── components/       # المكونات القابلة لإعادة الاستخدام (UI, Layout, Charts)
├── context/          # إدارة الحالة العالمية (State Management) عبر Context API
├── data/             # البيانات الثابتة (الأسئلة، الأوسمة)
├── hooks/            # Hooks مخصصة (تحميل البيانات، LocalStorage)
├── pages/            # الصفحات الرئيسية للتطبيق
├── services/         # خدمات الربط مع Gemini API
├── utils/            # دوال المساعدة، الحسابات، والتحقق من البيانات
└── __tests__/        # ملفات الاختبارات المنطقية
```

## دليل المستخدم (User Guide)
1. **التقييم**: ابدأ بالدخول إلى "التقييم المهني" وأكمل الاستبيان المكون من 15 سؤالاً.
2. **التوصيات**: بعد الإكمال، ستنتقل تلقائياً لصفحة التوصيات لمشاهدة مسارك الأمثل.
3. **الملف الشخصي**: قم بإضافة مشاريعك ودوراتك التدريبية لرفع مستواك وتحصيل أوسمة جديدة.
4. **السيرة الذاتية**: من صفحة الملف الشخصي، انقر على "عرض السيرة الذاتية" ثم "تحميل PDF" للحصول على وثيقة احترافية.
5. **السوق**: تابع صفحة "اتجاهات السوق" لتبقى على اطلاع بآخر التغيرات في الرواتب والمهارات.

## استكشاف الأخطاء وإصلاحها (Troubleshooting)
- **مشكلة في تحميل التوصيات**: تأكد من استقرار اتصال الإنترنت ومن صلاحية مفتاح Gemini API.
- **فقدان البيانات عند التحديث**: المنصة تستخدم LocalStorage لحفظ حالتك محلياً. تأكد من عدم مسح ذاكرة التخزين المؤقت للمتصفح.
- **عدم ظهور الرسوم البيانية**: تأكد من عدم وجود إضافات تمنع تشغيل JavaScript أو تمنع تحميل المكتبات الخارجية.
